## **Usage**
Compile using `make` and run `so_long` with the path to your map file:
```sh
$ ./so_long PATH_TO_THE_MAP_FILE.BER
```
For the bonus `make bonus` and run `so_long_bonus` as such:
```sh
$ ./so_long_bonus PATH_TO_THE_MAP_FILE.BER
```
